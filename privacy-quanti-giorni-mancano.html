Informativa Privacy

Informativa sul trattamento dei dati per l’app Quanti giorni mancano, sviluppata da Giuseppe Maiello - Pin Soft House.

Ultimo aggiornamento: 29 aprile 2026

In sintesi: l’app “Quanti giorni mancano” è pensata per funzionare principalmente sul tuo dispositivo. Gli eventi, le date, le note e le preferenze vengono salvati localmente sul dispositivo dell’utente.
📅 Countdown 🔔 Notifiche locali 💾 Backup 🔒 Nessun account obbligatorio

1. Titolare del trattamento

Il titolare del trattamento dei dati è:

Giuseppe Maiello - Pin Soft House
Email di contatto: pinsofthouse@gmail.com

2. Dati trattati dall’app

L’app permette all’utente di creare e gestire countdown per eventi, ricorrenze e date importanti. I dati inseriti possono includere:

  • nome dell’evento;
  • data dell’evento;
  • ora della notifica, se impostata;
  • eventuali note o descrizioni inserite dall’utente;
  • icona o emoji scelta per l’evento;
  • colore associato all’evento;
  • preferenza relativa all’attivazione o disattivazione della notifica;
  • impostazione di eventuale ricorrenza annuale.

Questi dati sono inseriti volontariamente dall’utente e servono esclusivamente per il funzionamento delle funzionalità dell’app.

3. Dove vengono salvati i dati

I dati degli eventi vengono salvati localmente sul dispositivo dell’utente. L’app non richiede la creazione di un account e non invia automaticamente i dati a server esterni gestiti dallo sviluppatore.

Se l’utente decide di esportare o condividere un backup, il file generato viene gestito tramite le funzioni di condivisione del dispositivo. In questo caso, l’utente sceglie volontariamente dove salvare o inviare il file.

4. Finalità del trattamento

I dati vengono utilizzati per:

  • mostrare i countdown degli eventi creati dall’utente;
  • calcolare quanti giorni mancano a una determinata data;
  • gestire eventi futuri, passati e ricorrenti;
  • mostrare promemoria e notifiche locali;
  • consentire la ricerca e il filtro degli eventi;
  • permettere l’esportazione e l’importazione dei backup.

5. Notifiche

L’app può richiedere il permesso di inviare notifiche. Le notifiche vengono utilizzate per ricordare all’utente gli eventi salvati nell’app.

L’utente può attivare o disattivare le notifiche per i singoli eventi direttamente dall’app, oppure modificare le autorizzazioni dalle impostazioni del dispositivo.

Le notifiche sono gestite localmente dal sistema operativo del dispositivo.

6. Backup, esportazione e importazione

L’app può permettere all’utente di esportare un file di backup contenente gli eventi salvati. Questo file può includere nomi eventi, date, orari, note, icone, colori e altre impostazioni associate.

L’utente è responsabile della conservazione e della condivisione del file di backup. Se il file viene inviato tramite email, app di messaggistica, cloud o altri servizi esterni, il trattamento dei dati dipenderà anche dalle condizioni privacy di tali servizi.

7. Dati condivisi con terze parti

L’app non condivide automaticamente i dati degli eventi con terze parti.

I dati possono essere condivisi con app o servizi esterni solo se l’utente sceglie volontariamente di esportare, salvare o condividere un backup tramite le funzioni del dispositivo.

8. Permessi richiesti

L’app può richiedere alcuni permessi necessari al corretto funzionamento:

  • Notifiche: per inviare promemoria relativi agli eventi salvati.
  • Archiviazione/File: ove necessario, per creare, esportare o importare file di backup.
  • Vibrazione: se usata dal dispositivo per accompagnare notifiche o feedback.

I permessi possono essere modificati o revocati in qualsiasi momento dalle impostazioni del dispositivo.

9. Pubblicità, analytics e tracciamento

L’app, nella configurazione descritta da questa informativa, non utilizza account obbligatori, sistemi di pubblicità, profilazione o strumenti di tracciamento analytics gestiti dallo sviluppatore.

Se in futuro venissero introdotti servizi di terze parti, questa informativa verrà aggiornata.

10. Conservazione dei dati

I dati rimangono salvati sul dispositivo finché l’utente non li elimina manualmente dall’app, cancella i dati dell’app o disinstalla l’applicazione.

I file di backup esportati restano sotto il controllo dell’utente e possono essere eliminati manualmente dal percorso in cui sono stati salvati.

11. Diritti dell’utente

Poiché i dati sono salvati principalmente sul dispositivo, l’utente può gestirli direttamente dall’app. In particolare può:

  • creare, modificare o eliminare eventi;
  • disattivare le notifiche per singoli eventi;
  • eliminare eventi passati;
  • cancellare i dati dell’app dalle impostazioni del dispositivo;
  • disinstallare l’app per rimuovere i dati locali associati.

Per richieste o chiarimenti è possibile contattare lo sviluppatore all’indirizzo email indicato in questa informativa.

12. Sicurezza

L’app è progettata per limitare il trattamento dei dati al minimo necessario per offrire le sue funzionalità. L’utente è invitato a conservare con attenzione eventuali file di backup esportati, poiché possono contenere informazioni personali inserite volontariamente negli eventi o nelle note.

13. Minori

L’app non è destinata specificamente alla raccolta di dati personali di minori. Gli utenti minorenni dovrebbero utilizzare l’app con il consenso o la supervisione di un genitore o di chi ne esercita la responsabilità.

14. Modifiche a questa informativa

Questa informativa può essere aggiornata in caso di modifiche all’app, alle sue funzionalità o ai servizi utilizzati. La versione aggiornata sarà resa disponibile attraverso la pagina dell’app o altri canali appropriati.
